# HarmonyOS 悬浮时钟应用 (FloatingClock)

一个在鸿蒙OS PC/平板设备上运行的桌面悬浮时钟小挂件应用，支持实时时钟、倒计时、秒表三种模式，以及多种主题切换，可自由拖动位置并始终显示在所有应用最前端。

## 功能特性
- ⏰ 实时时钟（精确到秒）
- ⏱️ 倒计时模式（可自定义1-60分钟，最后10秒红色提示）
- ⏲️ 秒表模式（精确到0.01秒）
- 🎨 6种主题可选（经典深色 / 清爽浅色 / 海洋蓝 / 日落橙 / 森林绿 / 极简黑白）
- 🪟 系统级悬浮窗，始终显示在所有应用最前端
- 🖐️ 支持拖动自由改变位置
- 💾 配置自动持久化

## 使用方式
1. 用 DevEco Studio 打开项目
2. 连接 HarmonyOS 6.0+ 设备
3. 编译并安装应用
4. 在主界面选择主题和模式，点击"显示悬浮窗"按钮

## 交互方式
- **单击**：启动/暂停倒计时或秒表；时钟模式下打开设置菜单
- **长按**：弹出设置菜单（切换主题 / 切换模式 / 显示日期 / 关闭悬浮窗）
- **拖动**：按住拖拽改变窗口位置

## 技术栈
- ArkTS + 声明式UI (ArkUI)
- 目标 SDK：HarmonyOS 6.0+
- 窗口管理：`@kit.ArkUI window` (WindowType.TYPE_FLOAT)
- 数据持久化：`@kit.ArkData preferences`
